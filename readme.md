## Introduction

This portfolio website is a reflection of my journey as a developer. It includes information about my skills, work experience, and projects. The site is designed to be both functional and visually appealing, showcasing my abilities in web development and design.

## Features

- **Homepage**: Introduction and overview of my skills and projects with buttons to explore more.
- **About Me**: A section with detailed information about my background and achievements.
- **Skills**: A comprehensive list of my technical skills, including programming languages and tools.
- **Experience**: A timeline showcasing my work experience and education.
- **Projects**: An overview of the projects I have worked on, with descriptions and links to GitHub repositories.
- **Contact Me**: A form to get in touch with me directly.

## Technologies Used

- **HTML**: For structuring the content on the web pages.
- **CSS**: For styling and layout of the website.
- **JavaScript**: For interactive features and dynamic content.
- **Typed.js**: For creating dynamic text animations.

## Installation

To run this project locally, follow these steps:

1. Clone this repository:
   ```bash
   git clone https://github.com/bhaumik-solanki/personal-portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd personal-portfolio
   ```

3. Open `index.html` in your preferred web browser to view the portfolio.

## Usage

Feel free to explore the different sections of the portfolio by clicking on the navigation buttons. You can also view the code and contribute by opening issues or making pull requests.
